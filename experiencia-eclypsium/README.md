# Experiencia Profesional – Eclypsium Inc.

## Rol: Práctica Profesional Supervisada (PPS) – Ingeniería Informática
**Empresa:** Eclypsium Inc., Córdoba Capital  
**Año:** 2025

Durante mi práctica profesional en Eclypsium trabajé en un equipo técnico de la empresa, participando en tareas reales de desarrollo, mantenimiento de herramientas internas, automatización de validaciones y soporte para aplicaciones web. Esta experiencia me permitió trabajar en entornos profesionales, usar metodologías de trabajo ágiles y contribuir directamente a proyectos utilizados por el equipo técnico.

---

# Tareas realizadas

### 1. Desarrollo y mantenimiento de ABRAM Linter
Herramienta interna utilizada para validaciones automáticas.

- Implementación de flags `--only` y `--exclude` para ejecución selectiva del linter.
- Correcciones relacionadas con archivos Yamale.
- Verificación de definiciones duplicadas.
- Ajustes y mejoras en funciones internas del linter.

### 2. Desarrollo web (Frontend + Backend)

- Implementación de un frontend simple para una aplicación FastAPI.
- Desarrollo de tests automatizados para la interfaz.
- Integración de HTMLX para componentes dinámicos.

### 3. Automatización y pruebas

- Creación de tests con Pytest para validar comportamientos.
- Participación en mejoras de pipelines internas.

### 4. Control de versiones y repositorios

- Uso diario de GitHub (branches, pull requests, revisiones).
- Corrección de problemas con Git LFS y actualización de `.gitattributes`.
