### Tablero de Control Estratégico para Automatización Industrial

Este proyecto consiste en el diseño e implementación de un tablero de control orientado a la medición del desempeño en procesos automatizados dentro del ámbito industrial. El objetivo principal es analizar la eficiencia de planta, detectar fallos y evaluar el impacto de la automatización en la competitividad del negocio.

### Objetivos del sistema

* Medir y mejorar la eficiencia operativa.
* Monitorear fallas en tiempo real.
* Analizar participación en el mercado (Market Share).
* Unificar métricas técnicas y estratégicas para la toma de decisiones gerenciales.

### Tecnologías utilizadas

* HTML, CSS y Bootstrap para la interfaz.
* JavaScript para el cálculo de KPIs.
* JSON para el manejo de datos.

### Principales indicadores implementados

El sistema calcula de forma automática:

* OEE (Disponibilidad × Performance × Calidad)
* Utilización de recursos
* Tasa de fallas
* Market Share

Además compara periodos históricos y visualiza tendencias operativas para facilitar la toma de decisiones.

### Resultados destacados

* OEE cercano al 80 %.
* Nivel mínimo de fallas (< 0.05 %).
* Evolución positiva de participación de mercado y productividad.

### Valor del proyecto

El tablero funciona como una herramienta integral para el monitoreo industrial y la mejora continua, permitiendo visualizar rendimiento, anticipar fallos y evaluar impacto comercial de la automatización en tiempo real.
