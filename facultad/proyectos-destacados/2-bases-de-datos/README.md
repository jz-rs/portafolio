# Base de Datos – Sistema de Gestión

## Descripción
Proyecto académico donde se diseñó, implementó y consultó una base de datos relacional,
aplicando buenas prácticas de modelado y SQL.

## Contenidos
- Diseño de base de datos (modelo E-R)
- Consultas SQL complejas
- Procedimientos almacenados y triggers

## Tecnologías
- SQL
- Modelo relacional
- MySQL

## Organización
Cada carpeta corresponde a una etapa del proyecto.
