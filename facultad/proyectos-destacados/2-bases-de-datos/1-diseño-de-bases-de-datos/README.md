# Diseño de Base de Datos – Foro de Posteos

Proyecto de diseño conceptual y lógico para un sistema de foro donde los usuarios pueden publicar posteos, dar like/dislike, seguir a otros usuarios y explorar temas.

## Contenido del proyecto
* Análisis de procesos (registro, posteos, interacciones, perfiles)
* Identificación de entidades y atributos
* Relaciones entre entidades
* Diagrama Entidad–Relación (DER)
* Descripción completa del modelo

## Entidades principales
* Usuarios
* Posteos
* Temas
* Likes
* Reposteo
* Usuarios seguidos

El archivo PDF incluye la documentación completa del diseño.
