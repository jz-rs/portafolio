# SQL – Consultas y Administración de Estructuras

Proyecto basado en una base de datos de electrodomésticos. Incluye creación de tablas, claves foráneas, constraints, migraciones y consultas avanzadas.

## Contenido del proyecto
* Script completo de creación del modelo relacional
* Conversión del esquema para permitir múltiples ventas por producto
* Migración de datos sin pérdida de información
* Eliminación y re-creación de constraints
* Consultas avanzadas

## Destacados técnicos
* Uso de `CHECK`, `FOREIGN KEY`, `PRIMARY KEY`, `UNIQUE`
* Migración con `INSERT ... SELECT`
* Modificaciones con `ALTER TABLE`
* Consulta que resume partes utilizadas por año de fabricación

Los archivos `.sql` contienen todo el desarrollo.
