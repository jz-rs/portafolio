# Desarrollo Frontend con React

## Descripción

Curso de Desarrollo Frontend orientado a la creación de aplicaciones web modernas, utilizando **React** como biblioteca principal. Se abordaron los fundamentos del desarrollo web, el uso de JavaScript, el desarrollo de aplicaciones con React, la integración con servicios externos, nociones de seguridad y despliegue de aplicaciones, complementando contenidos vistos en la formación universitaria.

## Temas del curso

### Introducción al desarrollo web

* Conceptos básicos de programación.
* Qué es el desarrollo web y cómo funciona.
* Rol del frontend dentro de una aplicación web.
* Relación entre frontend y backend.

### HTML y CSS

* Fundamentos de HTML.
* Estructura y semántica de documentos web.
* Fundamentos de CSS.
* Aplicación de estilos y diseño visual.
* Separación entre estructura y presentación.

### JavaScript

* Introducción al lenguaje JavaScript.
* Lógica del lado del cliente.
* Interacción con el contenido de la página.
* Manejo de eventos y comportamiento dinámico.

### React

* Fundamentos de React.
* Concepto de componentes.
* Componentes funcionales.
* Uso de props y estado.
* Renderizado y actualización de interfaces.
* Organización y reutilización de componentes.

### Desarrollo de aplicaciones con React

* Construcción de aplicaciones web usando React.
* Estructura de proyectos React.
* Manejo del flujo de datos.
* Desarrollo de interfaces dinámicas.

### Generación de código asistida por IA

* Uso de herramientas de inteligencia artificial como apoyo al desarrollo.
* Generación y análisis de código con asistencia de IA.
* Buenas prácticas en el uso de IA como herramienta complementaria.

### Despliegue de aplicaciones React

* Concepto de despliegue de aplicaciones web.
* Preparación de una aplicación React para producción.
* Publicación de aplicaciones frontend.

### Seguridad en aplicaciones web

* Conceptos básicos de seguridad en aplicaciones web.
* Buenas prácticas para el desarrollo seguro.
* Conciencia sobre riesgos comunes en aplicaciones frontend.

### Integración con servicios de terceros

* Consumo de APIs.
* Integración con servicios externos.
* Uso de datos dinámicos en aplicaciones frontend.

### Desarrollo avanzado en React

* Profundización en conceptos de React.
* Mejores prácticas en el desarrollo de aplicaciones React.
* Organización y mantenimiento de aplicaciones frontend complejas.

## Competencias adquiridas

* Comprensión del desarrollo frontend moderno.
* Capacidad para desarrollar aplicaciones web con React.
* Conocimiento del consumo de APIs y servicios externos.
* Nociones de despliegue y seguridad en aplicaciones web.
* Base sólida para continuar profundizando en desarrollo frontend.

* HTML
* CSS
* JavaScript
* React
* APIs
* Desarrollo Frontend
* Seguridad web
