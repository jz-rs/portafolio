# Análisis de Datos – Fundamentos

## Descripción

Curso introductorio de Análisis de Datos orientado a comprender cómo recolectar, analizar e interpretar datos para obtener información útil que apoye la toma de decisiones. Se trabajaron los conceptos fundamentales del análisis de datos y su aplicación en contextos reales, complementando contenidos vistos en la formación universitaria.

## Contenidos abordados

### Introducción al análisis de datos

* Qué es el análisis de datos.
* Importancia del análisis de datos en organizaciones y proyectos.
* Tipos de datos y fuentes de datos.
* Rol del analista de datos.

### Recolección y preparación de datos

* Obtención de datos desde distintas fuentes.
* Organización de datos.
* Limpieza de datos.
* Detección de errores, valores faltantes e inconsistencias.
* Importancia de la calidad de los datos.

### Análisis exploratorio

* Exploración inicial de los datos.
* Identificación de patrones y tendencias.
* Uso de estadísticas descriptivas.
* Interpretación de resultados.

### Visualización de datos

* Importancia de la visualización en el análisis.
* Tipos de gráficos.
* Selección de visualizaciones adecuadas según el objetivo.
* Comunicación clara de la información.

### Interpretación y toma de decisiones

* Análisis de resultados.
* Extracción de conclusiones a partir de los datos.
* Uso de datos para apoyar decisiones.
* Comunicación de insights de manera comprensible.

### Buenas prácticas

* Organización del trabajo de análisis.
* Documentación de resultados.
* Pensamiento crítico frente a los datos.
* Ética y responsabilidad en el uso de la información.

## Competencias adquiridas

* Comprensión del proceso completo de análisis de datos.
* Capacidad para interpretar información a partir de datos.
* Pensamiento analítico y crítico.
* Base sólida para profundizar en herramientas y técnicas de análisis de datos.

## Conceptos y herramientas

* Análisis de datos
* Estadística básica
* Visualización de datos
* Interpretación de información
* Toma de decisiones basada en datos
