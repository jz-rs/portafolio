# Introducción a la Programación con Python

## Descripción

Curso introductorio orientado a aprender los fundamentos de la programación utilizando Python. Se trabajaron los conceptos básicos necesarios para desarrollar programas simples, estructurados y legibles, haciendo foco en la lógica, el manejo de datos y el flujo de control.

## Contenidos vistos

### Fundamentos de programación

* Qué es la programación: concepto de input, procesamiento y output.
* Lenguajes de programación y propósito de Python.
* Buenas prácticas y prolijidad en el código.
* Uso de Google Colab como entorno de trabajo.

### Entrada y salida de datos

* Uso de `print()` para mostrar información.
* Uso de `input()` para ingresar datos por teclado.
* Conversión de tipos (`int()`, `float()`, `str()`).
* Manejo básico de errores de entrada.

### Variables

* Asignación de variables.
* Reglas para nombres de variables.
* Tipado dinámico en Python.
* Uso de variables numéricas y de texto.

### Tipos de datos

* Numéricos: `int`, `float`.
* Booleanos: `True`, `False`.
* Strings (texto).
* Conversión entre tipos (casting).
* Operaciones básicas con cada tipo.
* Precisión y redondeo de números decimales.

### Control de flujo

* Estructuras condicionales:

  * `if`
  * `if / else`
  * `if / elif / else`
* Uso de operadores lógicos y de comparación.
* Importancia de la indentación en Python.

### Bucles

* Bucle `while`:

  * Condiciones de repetición.
  * Prevención de bucles infinitos.
* Bucle `for`:

  * Uso de `range()`.
  * Recorridos controlados.
* Uso de bucles para validación de datos.

### Funciones

* Definición de funciones con `def`.
* Parámetros de entrada y valores de retorno.
* Uso de `return`.
* Diferencia entre imprimir y devolver valores.
* Reutilización de código.
* Combinación de funciones con estructuras de control.
* Ejemplos prácticos (suma, validación de contraseña).

### Strings

* Operaciones con strings:

  * Concatenación.
  * Longitud (`len`).
  * Búsqueda (`in`).
* Métodos:

  * `startswith()`
  * `endswith()`
  * `replace()`
* Indexado y slicing.
* Recorrido de strings con `for`.
* Manipulación y análisis de texto.

### Estructuras de datos

#### Listas

* Creación de listas.
* Acceso por índice.
* Modificación de elementos.
* Slicing en listas.
* List comprehension.
* Operaciones comunes:

  * `append()`
  * `extend()`
  * `remove()`
  * `pop()`
  * `sort()`
  * `max()` / `min()`
* Recorrido de listas con `for`.
* Listas con distintos tipos de datos.

### Manipulación de archivos

* Lectura de archivos de texto:

  * `open()`
  * `readlines()`
* Procesamiento de texto línea por línea.
* Limpieza de caracteres no deseados.
* Escritura de archivos:

  * Modos `r`, `w`, `a`.
  * Uso de `close()`.
  * Escritura con saltos de línea (`\n`).

### Diagramas de flujo

* Uso de diagramas para planificar programas.
* Identificación de:

  * Inicio
  * Proceso
  * Decisión
  * Fin
* Traducción de un diagrama de flujo a código Python.

## Trabajos prácticos

* Programas interactivos con ingreso de datos.
* Ejercicio de control de flujo con números pares e impares.
* Implementación de bucles para validación.
* Desarrollo de un programa basado en un diagrama de flujo.
* Manipulación y análisis de archivos de texto.
