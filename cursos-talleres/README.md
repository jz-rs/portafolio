# Cursos y talleres (2022–2025)

Esta carpeta reúne cursos y talleres que realicé como **refuerzo** y ampliación de mi formación universitaria en Ingeniería Informática.

## Cómo está organizado

- Cada curso/taller tiene su propia carpeta.
- Dentro de cada carpeta hay un `README.md` con:
  - Qué aprendí
  - Temas principales
  - Evidencias (certificado, apuntes y/o material)
  - En algunos casos, un **trabajo final integrador** (si aplica)
