# SQL Interactivo – Desafío Latam (Curso de Refuerzo)

## Descripción

Curso interactivo de **SQL y Bases de Datos Relacionales** realizado en la plataforma de **Desafío Latam**, orientado a comprender los fundamentos del lenguaje SQL y el modelo relacional. El curso se desarrolló mediante niveles progresivos con validación automática de consultas, y fue complementado con estudio teórico adicional y apuntes propios para afianzar conceptos.

Este curso fue realizado como **formación previa y de refuerzo** a los contenidos de Bases de Datos vistos posteriormente en la carrera universitaria.

## Modalidad

* Plataforma interactiva con explicación teórica.
* Resolución de consultas SQL con validación automática.
* Avance por niveles según resultados correctos.
* Certificados de progreso por cada bloque completado.
* Estudio complementario con material externo y apuntes propios.

## Contenidos abordados

### Fundamentos de bases de datos

* Qué es una base de datos.
* Para qué se utilizan las bases de datos.
* Ventajas del uso de bases de datos.
* Rol del sistema gestor de bases de datos (DBMS).

### Bases de datos relacionales

* Organización de la información en tablas.
* Filas (registros) y columnas (campos).
* Claves primarias (Primary Key).
* Claves foráneas (Foreign Key).
* Integridad de datos.
* Relaciones entre tablas.

### Modelado de datos

* Identificación de entidades.
* Definición de atributos.
* Relaciones entre entidades.
* Cardinalidades:

  * Uno a uno.
  * Uno a muchos.
  * Muchos a muchos.
* Diagramas entidad–relación (DER).
* Importancia del diseño previo a la implementación.

### Tipos de bases de datos

* Bases de datos relacionales.
* Bases de datos no relacionales.
* Diferencias generales entre ambos enfoques.
* Casos de uso comunes.

### Introducción a SQL

* Qué es SQL.
* Uso del lenguaje SQL para interactuar con bases de datos.
* Estructura general de una consulta SQL.

### Definición de estructuras

* Creación de tablas.
* Definición de columnas y tipos de datos.
* Tipos de datos más utilizados:

  * Numéricos.
  * Texto.
  * Fechas.
* Uso de claves primarias.
* Autoincremento.
* Restricciones básicas.

### Consultas básicas

* SELECT.
* WHERE.
* ORDER BY.
* LIMIT.
* Uso de condiciones y filtros.
* Ordenamiento de resultados.

### Consultas sobre datos agrupados

* GROUP BY.
* Uso de funciones de agregación.
* Análisis de datos por categorías.

### Funciones de agregación

* COUNT.
* SUM.
* AVG.
* MIN.
* MAX.
* Uso de funciones para análisis de información.

### Combinaciones y subconsultas

* Concepto de combinaciones entre tablas.
* Uso de relaciones para obtener información relacionada.
* Introducción a subconsultas.
* Comprensión del flujo de datos entre tablas.

### Manipulación de datos

* Inserción de registros.
* Edición de registros.
* Eliminación de registros.
* Importancia del cuidado al modificar información.

### Buenas prácticas

* Uso correcto de claves primarias y foráneas.
* Nombres claros para tablas y columnas.
* Consultas legibles y organizadas.
* Comprensión del impacto de las operaciones sobre los datos.
* Diseño consistente de la base de datos.

## Certificaciones obtenidas

* Consultas básicas en SQL.
* Consultas sobre datos agrupados.
* Combinaciones y subconsultas.
* Inserción, edición y eliminación de registros.
* Tablas y fundamentos de integridad de datos.

## Relación con la formación universitaria

Este curso funcionó como **base y refuerzo previo** a los contenidos formales de Bases de Datos vistos en la carrera, facilitando la comprensión del modelo relacional, el diseño de tablas y el uso práctico del lenguaje SQL.

## Competencias adquiridas

* Comprensión del modelo relacional.
* Capacidad para diseñar estructuras de datos simples.
* Lectura y escritura de consultas SQL.
* Análisis básico de datos desde una base de datos.
* Base sólida para profundizar en bases de datos avanzadas.
